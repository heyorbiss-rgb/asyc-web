Agregar una noticia

Esta página es solo para ti — no aparece en el menú del sitio. Completa los campos, genera el código y pégalo en tu index.html.

Importante: este sitio no tiene base de datos, así que el panel no “guarda” la noticia solo. Genera el código y tú lo pegas en index.html y subes el archivo a Netlify. Es el paso final, pero es rápido.

Código para pegar
Cómo pegarlo
  1. Copia el código de arriba.
  2. Abre index.html en VS Code.
  3. Presiona Ctrl/Cmd + F y busca: PEGA AQUÍ LAS NOTICIAS NUEVAS
  4. Pega el código en la línea de abajo de ese comentario (lo que pegues primero aparecerá primero en la web).
  5. Guarda (Cmd + S) y arrastra index.html a Netlify para publicar.

Para quitar una noticia vieja: en index.html borra el bloque que empieza con <a class="news" y termina en </a> de esa noticia.
